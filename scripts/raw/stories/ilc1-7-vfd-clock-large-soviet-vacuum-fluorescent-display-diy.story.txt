A clock project for the large VFD tubes ILC1-1/7 model made in the Soviet era. It uses a preprogrammed microcontroller & standard parts

The idea for this project began with a request from a kit builder who owned these vintage VFD tubes. These are very large vacuum fluorescent display tubes from the Soviet era. Here is the link to a partial description of the specifications of these tubes:Huge 7-Segment VFD ILC1-1/7 (ИЛЦ1-1/7).

After examining the specifications and studying the tubes, we developed a prototype PC board and driver and wrote some firmware to create the clock project. The kit we developed is DIY for anyone who can solder competently and identify some basic electronic components. Everything is standard sized with no surface mount parts so a novice solderer can do it. The chips including the preprogrammed microcontroller are set in sockets for easy replacement, and there are very few other components in the design.

The prototype PCBs are the "barebones" design which includes trace on the board only, no solder mask or silkscreen. The production run will include full coverage in solder mask and silkscreen, making it easy to assemble for a skilled kit builder.

The design is modular, so that seconds can be added. The board has a port for GPS or Wi-Fi module input as well.

Risks and challenges

The biggest challenge to this project was supplying the proper power for these large tubes, which draw about 200ma each on the filament only! We designed a robust "overkill" power supply for this, to drive the tubes and power the drivers.
